## Create .env file
```
MONGODB_URI=YOUR_URI_MONGODB
PORT=4001
GOOGLE_CLIENT_ID=YOUR_GOOGLE_CLIENT_ID
JWT_SECRET_KEY=IDontKnow
```

## Settup project
```
npm install
```

```
npm start
```
